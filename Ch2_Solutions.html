Deriving the upper bound: \[ \begin{align*} \frac{f(x)}{g(x|b)} &= \frac{\frac{1}{\sqrt{2\pi}}\exp\left(-\frac{x^2}{2}\right)}{\frac{1}{2b}\exp\left(-\frac{|x|}{b}\right)}\\[2ex] &= \sqrt{\frac{2}{\pi}}\;b\; \exp\left(-\frac{x^2}{2}\right) \exp\left(\frac{|x|}{b}\right)\\[2ex] &= \sqrt{\frac{2}{\pi}}\;b\; \exp\left(\frac{2|x| - x^2b}{2b}\right)\\[2ex] &=\left\{ \begin{array}{ll} \sqrt{\frac{2}{\pi}}\;b\; \exp\left(\frac{2 x - x^2b}{2b}\right) &\;\text{for}\; x\geq 0\\ \sqrt{\frac{2}{\pi}}\;b\; \exp\left(\frac{-2 x - x^2b}{2b}\right) &\;\text{for}\; x <0,\\ \end{array} \right. \end{align*} \] where \(b>0.\)
    First, consider the case of \(x\geq 0.\) Maximizing
    \[ \begin{align*} \exp\left(\frac{2x - x^2b}{2b}\right) & = \exp\left(\frac{1}{b}x - \frac{1}{2}x^2 \right) \end{align*} \] with respect to \(x\geq 0\) yields \[ \begin{align*} \exp\left(\frac{1}{b}x - \frac{1}{2}x^2 \right) \cdot \left(\frac{1}{b} - x \right) &\overset{!}{=} 0 \Leftrightarrow x=\frac{1}{b} \end{align*} \] Second, consider the case of \(x<0.\) Maximizing \[ \begin{align*} \exp\left(\frac{-2x - x^2b}{2b}\right) & = \exp\left(-\frac{1}{b}x - \frac{1}{2}x^2 \right) \end{align*} \] with respect to \(x < 0\) yields \[ \begin{align*} \exp\left(-\frac{1}{b}x - \frac{1}{2}x^2 \right) \cdot \left(-\frac{1}{b} - x \right) &\overset{!}{=} 0 \Leftrightarrow x=-\frac{1}{b} \end{align*} \] Thus \[ \begin{align*} \frac{f(x)}{g(x|b)} & \leq %\left\{ % \begin{array}{ll} % \sqrt{\frac{2}{\pi}}\;b\; \exp\left(\frac{2 \frac{1}{b} - \frac{1}{b^2} b}{2b}\right) &\;\text{for}\; x\geq 0\\ % \sqrt{\frac{2}{\pi}}\;b\; \exp\left(\frac{2 \frac{1}{b} - \frac{1}{b^2} b}{2b}\right) &\;\text{for}\; x <0\\ % \end{array} %\right.\\[2ex] \sqrt{\frac{2}{\pi}}\;b\; \exp\left(\frac{2 \frac{1}{b} - \frac{1}{b} }{2b}\right) \\[2ex] &= \sqrt{\frac{2}{\pi}}\;b\; \exp\left(\frac{1}{2b^2}\right) \end{align*} \] for \(x\in\mathbb{R}.\) The following code produces a plot of the graph of \[ \frac{f(x)}{g(x|b)} = \sqrt{\frac{2}{\pi}}\;b\; \exp\left(\frac{2|x| - x^2b}{2b}\right) \] and marks the maximizing arguments \(x=\pm 1/b\) for a given value of \(b>0:\)
b <- 1
myfun <- function(x, b){sqrt(2/pi) * exp( (2*abs(x) - x^2 * b)/(2 * b) )}
xx <- seq(-(1/b)*3, (1/b)*3, len = 500)
yy <- myfun(xx, b = b)
plot(x = xx, y = yy, type = "l")
abline(v=c(-1/b, 1/b), lty=2)

Exercise 4.

  1. Deriving the probability of accepting \(Y\) from a simulation \[ (Y,U)\sim\mathcal{U}\left(\left\{(y,u)|y\in \operatorname{supp}(f_X)\;\text{and}\;0\leq u\leq Mg(x)\right\}\right). \] Since we accept \(Y\) only if \(U\leq f_X(Y),\) \[ \begin{align*} P\left(\text{Accepting }Y\right) &=P\left(U\leq f_X(Y)\right), \end{align*} \] where \(U\sim\mathcal{U}\left[0,Mg(Y)\right].\)
    Standardizing the uniform distribution yields \[ \begin{align*} P\left(\text{Accepting }Y\right) &=P\left(\frac{U}{Mg(Y)}\leq \frac{f_X(Y)}{Mg(Y)}\right),\\[2ex] \end{align*} \] where dividing by \(Mg(Y)\) is allowed, since \(Mg(y)>0\) for all \(y\in\operatorname{supp}(f_X).\)
    Now, using that \[ \frac{U}{Mg(Y)} \sim\mathcal{U}\left[0,1\right], \] we have \[ \begin{align*} P\left(\text{Accepting }Y\right) &=\int_{-\infty}^{\infty}\int_0^{\frac{f_X(y)}{Mg(y)}} 1 \; du\; g(y) \;dy\\[2ex] &=\int_{-\infty}^{\infty}\;\;\;\frac{f_X(y)}{Mg(y)}\;\; g(y) \;\;dy\\[2ex] &=\frac{1}{M}\int_{-\infty}^{\infty}\;f_X(y) \;\;dy\\[2ex] &=\frac{1}{M} \end{align*} \]
  2. Showing that \(M\geq 1:\) \[ \begin{align*} f_X(x) & \leq M g(x)\quad \text{for all }\quad x\in\operatorname{supp}(f_X)\\[2ex] \int f_X(x) dx & \leq \int M g(x)dx = M \int g(x)dx\\[2ex] 1 & \leq M \cdot 1\\[2ex] \end{align*} \]
